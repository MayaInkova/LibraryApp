LibraryApp


LibraryApp е приложение за управление на книги, състоящо се от три основни компонента:

Frontend (HTML/JavaScript) – Потребителски интерфейс за управление на книги.

Backend (Java/Spring Boot) – REST API, което обработва заявките за добавяне и извличане на книги.

C# клиент – Конзолно приложение, което комуникира със сървъра чрез HTTP заявки.



API Документация

Метод               URL                     Описание

GET                 /books               Връща списък с всички книги

POST                /books               Добавя нова книга
